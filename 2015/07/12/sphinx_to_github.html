Sphinx-to-github

sphinx-to-github - Sphinx extension & script to prepare Sphinx html output for github pages.

Некоторые генераторы статических сайтов используют sphinx. При попытке выложить статический сайт на Github pages столкнулся с тем, Github не работает с каталогами, которые начинаются с “_”. Утилита Sphinx-to-github при генерации сайта, адаптирует сайт под Github Pages удаляя символы подчеркивания “_” везде где нужно.

Как использовать:

  • Скачать файл sphinxtogithub.py.
  • Разместить в корне сайта с исходниками.
  • В conf.py добавить
import sys
import os
sys.path.append(os.getcwd())
import sphinxtogithub
...
# Добавить 'sphinxtogithub' в массив с другими расширениями
extensions = ['sphinxtogithub']
