Tinkerer is a blogging engine/static website generator powered by Sphinx. It allows blogging in reStructuredText format, comes with out-of-the-box support for post publishing dates, authors, categories, tags, post archive, RSS feed generation, comments powered by Disqus and more. Tinkerer is also highly customizable through Sphinx extensions.

Утановка

::
pip install tinkerer

Из зависимостей понадобится установить python-lxml.

::
apt-get install python-lxml
